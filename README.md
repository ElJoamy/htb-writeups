# HTB Write‑Ups by ElJoamy

These are my personal notes and walkthroughs for Hack The Box content. There are many ways to solve each target; what you will find here is one practical path that worked for me, with commands, reasoning, and verifiable outputs preserved for learning and reproducibility.

## Repository Structure

- [Machines](./Machines/README.md)
  - Standalone HTB machines.
  - Each write‑up focuses on clear recon, exploitation, and post‑exploitation steps.
- [ProLabs](./ProLabs/README.md)
  - Multi‑host or AD‑centric environments from HTB Pro Labs.
  - Write‑ups emphasize attack path context, lateral movement, and domain escalation.

Directory layout:
- ./[Machines](./Machines/README.md) — individual machines
- ./[ProLabs](./ProLabs/README.md) — pro lab environments

## How to Use This Repository

- Start from the index file in each folder to see available content.
- Open an individual write‑up to follow step‑by‑step instructions.
- Commands and tool outputs are kept intact to make it easy to replicate results.
